# README

## 開発言語
* Ruby version: 3.0.1
* Rails 6.0.3

## 実行手順
* $ git clone git@github.com:Roatgithub/go_to_kintai.git
* $ cd go_to_kintai
* $ bundle install
* $ yarn install --check-files
* $ rails db:create
* $ rails db:migrate
* $ rails db:seed
* $ rails s
